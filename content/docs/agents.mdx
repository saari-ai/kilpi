## Skills

Three Claude Code skills implement the Kilpi methodology, each referencing sections of these docs as its source of truth.

| Skill | When activated | Reads from |
|-------|---------------|------------|
| <span style={{whiteSpace: 'nowrap'}}>kilpi_docs</span> | Maintaining the Kilpi documentation structure itself | Conventions, page templates, references |
| <span style={{whiteSpace: 'nowrap'}}>kilpi_org</span> | Applying Kilpi at the organisational level | All 6 functions, Template (Org), [Taskflow](/docs/examples/taskflow) example |
| <span style={{whiteSpace: 'nowrap'}}>kilpi_component</span> | Applying Kilpi to a specific product or component | Identify, Protect, Detect, Template (Component), [Taskflow](/docs/examples/taskflow) example |
