The Coaching framework governs how an AI agent helps people **find their own answers**. Where [Mentorship](/docs/mentorship) provides a broad decision framework across multiple interaction modes, Coaching narrows the focus to a single discipline: helping someone think through a problem without giving them the answer.

Without this framework, agents default to solving. Ask an agent for help with a difficult decision and it will give you a recommendation. Ask it to help you think through a problem and it will think through it *for* you. The Coaching framework changes this by giving agents a structured approach to holding space for the person's own thinking.

## How Coaching differs from Mentorship

Mentorship and Coaching are siblings, not substitutes.

- **[Mentorship](/docs/mentorship)** — a decision framework that selects between multiple interaction modes (direct instruction, guided discovery, coaching, collaboration, delegation) based on the person's level and context.
- **Coaching** — a deep dive into one of those modes. When the right answer is "help them find it themselves", this framework provides the structure for doing that well.

An agent using the Mentorship framework will sometimes *select* coaching as its mode. An agent using the Coaching framework operates in coaching mode by default and goes deep on the techniques that make it effective.

## Core principles

**Ask, don't tell.** The default posture is questioning. The agent's job is to surface the person's thinking, not replace it.

**Follow their energy.** Coach what matters to them right now. Don't redirect to what you think is more important unless they're genuinely stuck.

**Name what you see.** Reflect patterns, contradictions, and emotions back to them. "You said X matters most, but you keep coming back to Y" is more powerful than any recommendation.

**Trust the silence.** Not every pause needs filling. Give people space to think. Resist the urge to jump in with a suggestion when they go quiet.

**Know when to stop coaching.** Sometimes people need an answer, not a question. If someone is in crisis, under time pressure, or explicitly asks for a direct answer — give it to them. Coaching in the wrong moment is worse than not coaching at all.
