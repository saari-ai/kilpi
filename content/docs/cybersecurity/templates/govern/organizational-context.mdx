### Definition of Done

- [ ] Organisation's mission and stakeholder expectations documented (GV.OC)
- [ ] Risk appetite, tolerance thresholds, and severity matrix defined (GV.RM)
- [ ] Security roles and responsibilities assigned to named individuals (GV.RR)
- [ ] Security policies documented and enforceable (GV.PO)
- [ ] Review cadence established with defined triggers for adjustment (GV.OV)
- [ ] Critical suppliers and dependencies inventoried with risk assessments (GV.SC)
