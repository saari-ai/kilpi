fiksu is a curated library of skills that help AI agents build, secure, and ship software. Each skill set gives agents the structured knowledge they need to operate effectively across security, architecture, and product disciplines.

## Skill Sets

- **[Kilpi](/docs/kilpi)** — Security Skills for securing AI generated applications and systems.
- **[Runko](/docs/runko)** — Architecture Skills for structuring AI generated applications and systems.
- **[Varma](/docs/varma)** — Product Skills for finding and validating products and features.
