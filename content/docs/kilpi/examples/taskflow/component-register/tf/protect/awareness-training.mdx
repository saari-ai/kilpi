### PR.AT-01: General awareness

| Topic | Audience | Delivery | Cadence |
|---|---|---|---|
| OWASP Top 10 overview | All developers | Team session + self-study materials | Annually, refreshed when OWASP updates |
| Secure coding practices (parameterised queries, output encoding, schema validation) | All developers | PR review checklist + onboarding walkthrough | Onboarding + continuous via PR reviews |
| Secret handling (never in code, logs, or client bundles) | All developers | Onboarding briefing + CI enforcement | Onboarding + continuous via CI |
| Phishing and social engineering awareness | All team members | Short video or article | Annually |
| Incident response overview (who to contact, how to escalate) | All team members | Onboarding briefing + incident response runbook | Onboarding + refreshed after each incident |
| Data classification policy | All team members | Written policy + onboarding walkthrough | Onboarding + annually |

### PR.AT-02: Specialized role training

| Role | Training topics | Delivery | Cadence |
|---|---|---|---|
| Security lead | NIST CSF 2.0 framework updates, threat modelling methodologies, incident command | External training / conference, self-study | Annually |
| Infrastructure owner | AWS security best practices, IAM policy design, VPC configuration, CloudTrail analysis | AWS training resources, hands-on labs | Annually |
| Incident responder (on-call) | Incident response playbook walkthrough, tabletop exercises, containment procedures | Team exercise + tabletop simulation | Quarterly |
| Dependency maintainer | Supply chain attack patterns, CVE triage, Dependabot configuration | Self-study + team knowledge sharing | Semi-annually |
