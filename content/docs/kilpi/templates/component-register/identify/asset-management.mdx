### Definition of Done

- [ ] All assets registered with IDs (`SA-{component}-*`) and classified by criticality and data sensitivity (ID.AM)
- [ ] Data flows documented with trust boundaries marked (ID.AM-03)
- [ ] STRIDE applied to every data flow crossing a trust boundary (ID.RA-03)
- [ ] All identified threats scored using the severity matrix (ID.RA-04)
- [ ] Risk responses (mitigate, accept, transfer, avoid) chosen for every Medium severity and above finding (ID.RA-06)
